TeamBridge Japan — Roles & Features

Last audited: 2026-05-16 · Based on direct source code analysis

最終監査: 2026年5月13日 · ソースコードの直接調査に基づく

7 roles in the system. Role is stored in profiles.role. Admin panel is granted to GM, Headcoach and Organisation Staff. The is_super_admin flag (separate DB column) is required to assign or change roles. システムには7つのロールがあります。ロールは profiles.role に保存されます。管理パネルはGM・ヘッドコーチ・組織スタッフに付与されます。ロール変更には is_super_admin フラグが必要です。

Feature機能
Player-only auto-triggered forms. Each popup appears based on specific conditions — no manual navigation required. Data is restricted to the coaching/medical circle (Organisation Staff excluded). 選手専用の自動トリガーフォームです。特定の条件に基づき自動表示 — 手動ナビゲーション不要。データはコーチング・医療スタッフのみ閲覧可能（組織スタッフを除く）。
The Dashboard adapts to the user's role — cards are shown or hidden based on permissions. All data is fetched in a single parallel Promise.all on load. Numbers below refer to the original improvement proposals (#1–#7). ダッシュボードはロールに応じて表示が変わります。権限に基づきカードを表示・非表示。データはロード時に並列取得。以下の番号は元の改善提案（#1〜#7）に対応します。
